# CaseConvertor

A modern, fast, and privacy-focused online text case conversion utility.

## Features

- **Multiple Case Formats**: Sentence case, lower case, UPPER CASE, Capitalized Case, aLtErNaTiNg cAsE, Title Case, iNVERSE cASE, and slug-case.
- **Real-time Statistics**: Instantly track character count, word count, and line count.
- **Privacy First**: All processing happens locally in your browser. Your data never leaves your device.
- **Modern UI**: Clean, responsive design with full Dark Mode support.
- **Performance Optimized**: High Lighthouse scores for Core Web Vitals (LCP, FCP).

## Built With

- **React 19**
- **Tailwind CSS**
- **ESM.sh** (for dependency management)
- **Google AI Studio**
